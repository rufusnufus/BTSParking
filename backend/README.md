# Backend

This is the backend server written on FastAPI and using Postgres as a database.

1. Go to backend directory: 
    ```bash
    cd backend
    ```

2. To build the backend Dockerfile: 
    ```bash
    DOCKER_BUILDKIT=1 docker build -t backend .
    ```
